# Quiz App Modern

A modern and interactive quiz application built with **React**, **Vite**, and styled using **Material UI** and **Mantine UI**. The app provides a smooth user experience with animations powered by **Framer Motion** and **GSAP**.

---

## 🚀 Features

* 🎯 Play quizzes with multiple-choice questions
* 📊 Score tracking system
* 🎨 Modern UI with **MUI** and **Mantine** components
* ⚡ Lightning-fast build with **Vite**
* 🔄 Client-side routing with **React Router DOM**
* ✨ Smooth animations using **Framer Motion** and **GSAP**

---

## 📂 Project Structure

```
quiz-app/
├── index.html
├── package.json
├── vite.config.js
├── src/
│   ├── components/   # Reusable UI components (QuestionCard, Navbar, etc.)
│   ├── data/         # Quiz data in JSON format
│   ├── pages/        # Different pages (Home, Quiz, Results, etc.)
│   ├── App.jsx       # Main application file
│   └── main.jsx      # React entry point
```

---

## 🛠️ Tech Stack

* **Frontend:** React 18, Vite 5
* **UI Libraries:** Material UI, Mantine UI
* **Animations:** Framer Motion, GSAP
* **Routing:** React Router DOM

---

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/quiz-app.git
   cd quiz-app
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run development server:

   ```bash
   npm run dev
   ```

4. Build for production:

   ```bash
   npm run build
   ```

5. Preview production build:

   ```bash
   npm run preview
   ```
